# 문제해결방식검사 — 검사 개요

## 1. 검사 정의

CRATA 문제해결방식검사는 사람이 **문제를 어떻게 이해하고 해결하며, 어떤 조직·직무·팀 구조에서 자신의 역량을 발휘하는지**를 분석하는 문제해결방식·환경적합성 검사다.

## 2. 역량 구조

역량은 4가지이며, 각 역량은 보통 **인지**와 **행동** 슬롯으로 나뉜다.

### 2-1. 중심역량
- **정의**: 사람의 인지와 행동의 중심을 잡아주는 환경조건을 찾아 삶의 뿌리 역할을 하는 역량.
- **핵심**: 무엇이 결핍되면 멘탈이 흔들리고, 무엇이 채워지면 마음이 안정되어 강점을 쓸 수 있는 바탕(심리적 안정감·환경적 자신감)이 되는가.
- 인지 / 행동 = 각각 조직 10유형 중 1개.

### 2-2. 핵심역량
- **정의**: 현실 문제를 해결하며 긍정적 결과를 만드는 행동 패턴. 반복되며 패턴이 되고 인정·성취감으로 이어진다.
- 인지 = 문제를 만났을 때 자동으로 먼저 작동하는 관점 (조직 10유형 중 1개).
- 행동 = 현실 조건에서 실제로 움직이고 해결하는 방식 (조직 10유형 중 1개).

### 2-3. 성장역량
- **정의**: 자기 이해를 통해 기존 반복 패턴을 인지하고, 그 반복을 넘는 다른 행동을 선택하게 만드는 역량.
- **인지 = 행동동기검사의 고유 기준 10조합 중 1개.** (조직 10유형이 아님 — 주의)
- 행동 = 기존 패턴을 넘는 선택적 행동 (조직 10유형 중 1개).

### 2-4. 잠재역량
- **정의**: 선택은 들어가지만 아직 구체적 형태로 정해지지 않은 가능성의 역량.
- **표기 수준**: 10유형 세부가 아니라 상위 5방식 수준에서만 나타난다.
- **개수 규칙**: 최소 1개, 최대 4개. **4개일 때는 자기효능감향상방식 포함 불가.**

## 3. 역량 배치 규칙

| 역량 자리 | 들어가는 값 |
|---|---|
| 중심역량 인지 | 조직 10유형 중 1개 |
| 중심역량 행동 | 조직 10유형 중 1개 |
| 핵심역량 인지 | 조직 10유형 중 1개 |
| 핵심역량 행동 | 조직 10유형 중 1개 |
| 성장역량 인지 | **행동동기검사 고유 기준 10조합 중 1개** |
| 성장역량 행동 | 조직 10유형 중 1개 |
| 잠재역량 | 상위 5방식 수준 (정보처리·능력표현·목표실행·목적성취·자기효능감향상) |

## 4. 헷갈리는 개념 비교

### 행동방식 4축 vs 조직 10유형
- 행동방식 4축 = 목적성취·정보처리·능력표현·목표실행 (8유형).
- 조직 10유형 = 위 8유형 + 자기효능감향상방식 2유형(경쟁형·비교형).
- **자기효능감향상방식은 역량 자리에는 들어갈 수 있으나 행동방식 4축 조합에는 포함하지 않는다.**
- **표기 노트**: "자기효능감향상방식" 이름은 **효능감 표기로 복귀 확정** (2026-05-27 후속 녹음 번복 — "효능감 맞는데"). 하위 유형 경쟁형·비교형 확정.

### 성장역량 인지 vs 성장역량 행동
- 인지 = 행동동기검사 고유 기준 10조합 중 1개.
- 행동 = 조직 10유형 중 1개.

### 잠재역량 vs 4축 "둘 다 없음"
- 잠재역량 = 상위 방식 수준의 선택적 가능성.
- 4축의 "둘 다 없음" = 해당 축의 세부유형이 나타나지 않는 상태.
