# 행동방식 4축

조직 10유형 중 8개가 속하는 상위 분류이자, 16행동유형·256상태조합을 산출하는 토대.
각 축(방식)에는 유형이 둘씩 들어 있고, 16유형은 각 축의 **존재 여부**(있음/없음)로 산출된다(`조합원리.md`).
각 방식과 그 안의 유형 정의는 `유형_조직10유형.md` 참조.

> **자기효능감향상방식(경쟁형·비교형)은 4축이 아니다.** 16유형 산출에 들어가지 않으며, 역량 자리에만 나타난다.
> 방식 차원의 설명이 필요해지면 의사결정방식검사(잠정명, 구 집단검사) 쪽에 두고 거기서 참조한다(이 검사 범위 밖).
> **표기 노트**: "자기효능감향상방식" 이름은 **효능감 표기로 복귀 확정** (2026-05-27 후속 녹음 3개 — "효능감 맞는데"로 번복, `decisions/log/2026-05-27_plaud_자기효능감검사_방향재설정.md`). 하위 유형 경쟁형·비교형 확정. 검사 자체가 자기효능감검사(잠정명)로 독립 예정 — 지식 디렉토리 신설 시 이전.

## 항목 틀
각 방식은 아래 세 가지로 적는다. (각 방식에는 유형이 둘씩 있다 — 예: 목적성취방식 = 과제형·관계형)
- **정의(공통 핵심)**: 이 방식이 한마디로 무엇을 하는 방식인가. 두 유형이 공통으로 가진 성격.
- **갈리는 결**: 같은 방식이라도 두 유형 중 어느 쪽이냐에 따라 갈리는 차이.
- **있을 때 / 없을 때**: 이 방식이 있으면 무엇이 더해지고, 없으면(공백) 무엇이 빠지는가.

> 이 세 가지가 16유형·256조합 설명의 토대가 된다.

---

## 목적성취방식 (O · 과제형 / 관계형)
- **정의(공통 핵심)**: 일에 목적·의미·책임의 분명함이 있어야 동력이 생기는 방식. 무엇을 위해, 누구를 위해 하는지가 잡혀야 움직인다.
- **갈리는 결**: 과제형(과업 완수·역할·규정·절차 자체에서 의미를 찾음) ↔ 관계형(사람의 인정·신뢰·존경에서 의미를 찾음). 목적을 *일·과업*에 두느냐 *사람·관계*에 두느냐.
- **있을 때 / 없을 때**: 있으면 → 일의 이유·목적·책임소재를 먼저 세우고 방향을 정함. O 공백이면 → 목적·이유를 따지기 전에 정보·방법·실행으로 바로 감(빠르나 "왜 하는가"가 약할 수 있음).

## 능력표현방식 (P · 설계형 / 기술형)
- **정의(공통 핵심)**: 자신의 능력과 기술, 방법을 써서 문제를 푸는 방식. "어떻게 풀 것인가"를 방법·구조·전문성으로 만들어낸다.
- **갈리는 결**: 설계형(넓게 — 전체 구조를 짜고 시스템화·일반화) ↔ 기술형(깊게 — 한 분야를 파고들어 전문성·구현). 능력을 *구조의 넓이*로 쓰느냐 *전문성의 깊이*로 쓰느냐.
- **있을 때 / 없을 때**: 있으면 → 문제에 맞는 방법·구조·기술을 스스로 만들어 적용. P 공백이면 → 고유한 방법·설계 없이 주어진 방식으로 처리(목적·실행이 있어도 "어떻게"가 약할 수 있음).

## 목표실행방식 (A · 전술형 / 전략형)
- **정의(공통 핵심)**: 목표를 향해 실제로 움직여 결과를 만들어내는 방식. 판단을 실행으로 옮기고 추진한다.
- **갈리는 결**: 전술형(지금 — 눈앞의 문제를 자율적으로 즉시 처리, 현재 보상) ↔ 전략형(나중 — 미래·경력·전체 맥락에 연결해 장기로 추진). 실행의 시계(視界)를 *지금*에 두느냐 *미래*에 두느냐.
- **있을 때 / 없을 때**: 있으면 → 판단을 실행으로 옮겨 결과를 만듦. A 공백이면 → 목적·방법·정보를 갖춰도 실제 추진·완수로 옮기는 힘이 약할 수 있음(시작 전 단계에 머묾).

## 정보처리방식 (S · 직관형 / 경험형)
- **정의(공통 핵심)**: 외부 정보를 받아들여 자기 것으로 만들고 문제 해결의 근거로 삼는 방식. 정보가 처리돼야 다음으로 나아간다.
- **갈리는 결**: 직관형(이해가 먼저 — 맥락·멘토·선행정보로 이해된 뒤 움직임) ↔ 경험형(경험이 먼저 — 직접 해보며 몸으로 익힘). 정보를 *이해로* 처리하느냐 *경험으로* 처리하느냐.
- **있을 때 / 없을 때**: 있으면 → 판단 전에 정보를 모으고 살펴 근거를 갖춤. S 공백이면 → 정보 수집·검토 없이 바로 목적·방법·실행으로 건너뜀(빠르나 근거가 얕을 수 있음).

## 동기성향 × 4축 매핑

> **두 검사 사이의 연결 자료.** 행동동기검사의 동기성향 5종(성장·발산·수확·축적·균형)이 각각 중심일 때, 본 검사 4축의 각 자리에 어떤 동기성향이 들어가는지를 정의한다. 근거: 2026-05-27 회의 도표(대표님 손글씨, 5개 중심형 각각의 4사분면 매핑).

### 4축의 자리 = 문제 해결의 4단계

| 자리 | 4축 | 4단계 의미 |
|---|---|---|
| 위 | 목적성취방식 (O) | **정의/준비** — 무엇을 위해 하는가의 방향이 잡히는 자리 |
| 왼 | 정보처리방식 (S) | **시작** — 외부 정보·근거가 들어와 움직임이 시작되는 자리 |
| 오 | 능력표현방식 (P) | **과정** — 능력·방법으로 풀어내는 자리 |
| 아래 | 목표실행방식 (A) | **결과** — 결과를 만들어내는 자리 |

### 매핑표 (중심형 × 4자리 → 동기성향)

| 중심형 | 위 (정의/준비)<br>목적성취 | 왼 (시작)<br>정보처리 | 오 (과정)<br>능력표현 | 아래 (결과)<br>목표실행 |
|---|---|---|---|---|
| **성장형** | 수확형 | 축적형 | 발산형 | 균형형 |
| **발산형** | 축적형 | 성장형 | 균형형 | 수확형 |
| **수확형** | 발산형 | 균형형 | 축적형 | 성장형 |
| **축적형** | 균형형 | 수확형 | 성장형 | 발산형 |
| **균형형** | 성장형 | 발산형 | 수확형 | 축적형 |

### 관찰된 순환 패턴 (참고)

기준 순환순서 **균형 → 성장 → 수확 → 발산 → 축적 → 균형**을 1-2-3-4-5로 두면, 각 자리는 중심형을 기준으로 다음과 같이 회전한다.

- **위 (정의/준비) = +1** / **아래 (결과) = -1**
- **오 (과정) = +2** / **왼 (시작) = -2**

5종 도표 모두 이 패턴과 정확히 일치한다. 이 순환순서 자체가 어떤 검사 원리(예: 동기성향 간의 거리·관계)에서 비롯됐는지는 추가 확정 필요(보류).

### 회의록(2026-05-27) 발언 근거

- *"균형형이 목적에 해당된다"* → 균형형(B) 중심 시 위(목적성취방식) = 성장형. 그래서 "목적에 성장형이 있다" → 미래지향적 사고가 자연스럽게 나옴
- *"B형들한테는 그 위에가 성장형"* → 도표의 균형형 중심과 정확히 일치
- *"능력 안에서는 너는 수확형"* (전사본엔 "수학형"으로 잘못 표기) → 균형형 중심 시 오(능력표현방식) = 수확형
- *"4사분면 중에 1,2 사분면 위로 더 높게 가 있잖아"* → 그래프 사분면에서 위쪽이 높음 = 목적성취방식이 강함 = 그 사람의 목적 자리에 +1 회전된 동기성향이 작동

### 자기효능감향상방식(경쟁형·비교형)

본 매핑에 포함되지 않는다. 4축이 아니므로 16유형·256조합 산출과 분리되어 있으며, 의사결정방식검사(잠정명, 구 집단검사)에서 다룬다.
