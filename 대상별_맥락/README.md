# 대상별 맥락

각 대상층(중학생·고등학생·대학생·직장인·부모·보호자·교사·교육자)의 **구체적 고민·맥락·자주 만나는 의뢰 패턴**을 누적하는 폴더. 일반론·추상 카테고리가 아니라 *"그 대상이 실제로 머리 속에 굴리는 것"*을 모은다.

## 누가 쓰는가

- **기획담당**: 의뢰의 대상층에 맞는 구체 고민을 매핑해 일반론을 피한다.
- **상담담당** (추후): 상담 사례 작성·해석 시 참조.

## 어떻게 누적하는가

자동 수정 X. 직원이 의뢰 처리 중 빈틈·새 패턴 발견 → 출력의 `[자산 보강 제안]` 슬롯으로 보고 → 메인 Claude가 대표님 컨펌 받음 → 반영. 모든 변경은 `decisions/log/`에 근거 기록.

## 각 .md 파일 구조

- **전형적 고민**: 구체 항목 5~10개. 추상 카테고리("자기이해 부족") 금지.
- **맥락·환경**: 그 대상의 일상·압박·관계 환경.
- **자주 만나는 의뢰 패턴**: 어떤 형태로 의뢰가 자주 들어오는지.

## 처음 비어있어도 됨

실전에서 부딪힌 부분만 누적. 미리 채우려 하지 않는다. 처음에 일부 항목이 보이는 건 대표님이 시드해주신 항목들이며, 모두 `초안` 상태로 검증·확장 대상이다.
