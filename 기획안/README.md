# 기획안

CRATA 검사를 활용한 프로그램·워크숍·교육 기획안의 누적 폴더.

## 무엇이 들어가는가

- 기획담당이 작성한 기획안 초안 (승인 후 보관)
- 실제 진행 후 결과 메모 (선택)

## 파일 명명

`YYYY-MM-DD_대상_주제.md` (예: `2026-05-23_직장인_시작어려움_워크숍.md`)

## 기획담당과의 관계

- 새 의뢰가 들어오면 기획담당은 이 폴더의 **과거 기획을 먼저 본다** — 같은 대상층·목적의 선례 참조.
- 기획에서 얻은 **일반화 가능한 원리**는 `decisions/log/`에 별도 기록. 누적되면 `rules/기획_원칙.md`로 승격 후보.
- 관점 틀기로 인접 가치를 살린 사례는 특히 잘 기록해두면 다음 의뢰에서 패턴 인식에 도움.
