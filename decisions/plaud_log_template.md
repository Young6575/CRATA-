# PLAUD 녹음 처리 기록 템플릿

> 실제 기록은 `decisions/log/YYYY-MM-DD_plaud_제목.md` 형식으로 저장한다.
> 이 파일은 템플릿 참고용이다.

---

## 기록 형식

```markdown
# PLAUD 녹음: [녹음 제목]

- **녹음 ID**: plaud_xxxxxxxx
- **녹음 일시**: YYYY-MM-DD HH:MM
- **처리 일시**: YYYY-MM-DD
- **참여자**: (해당 시)

## 핵심 내용 요약

(전사본에서 뽑은 핵심 내용 3-5줄)

## CRATA 작업 연결

| 작업 | 대상 파일 | 처리 여부 |
|------|----------|----------|
| (예) 조합설명 수정 논의 | 지식/조직검사/조합설명/... | 완료 / 보류 |

## 보류 항목

(이번에 처리 못 한 것, 다음 세션에서 이어갈 것)
```
